# CrossfitToolkit — Low-Fidelity Wireframes (Mobile-First)

## Design Principles
- Mobile-first, max-width ~430px for phone view
- Dark theme with red accent (#dc2626)
- Large touch targets (min 44px)
- Clear visual hierarchy
